## 关于键盘

- 在官方控制台里面只能切换开关；
- 使用键盘快捷键可以切换 3 挡：关闭，低亮度，高亮度；

## 寄存器定义

实际在 EC 1932 寄存器里面，手动控制有 5 挡可选：

  - 高位的四个字节，BIT4 在设置的时候必须为 1。
  - 五个挡位从 000 到 100，但是其实亮度从最亮到最暗，是从 010 100 001 011 000（关闭）这样排列的，如果你尝试用按键切换，那么他只会使用 001 和 010 分别作为暗和亮两档。

对应表：

|亮度等级|Bit7-Bit5|对应在键盘快捷键|
|---|---|---|
|0|000|关闭|
|1|011|比 Dim 低|
|2|001|Dim|
|3|100|比 Bright 低|
|4|010|Bright|


尽量不要切换到 1 和 3，此时背光快捷键切换会导致 EC 的位错乱（Bit 按二进制序列从 000-111），而且不能反映出实际的亮度，切回 0 挡即可拉回到正常状态。
